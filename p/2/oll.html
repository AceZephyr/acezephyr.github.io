What is OLL?

OLL is the second step of the Varasano method, where you orient the last layer.

Note that this OLL page is specifically for the Varasano method. Some of the algorithms (denoted with a *) swap pieces on the first layer, making them unusable for an LBL method.

So if you are using LBL and not Varasano, make sure to avoid those algs.
